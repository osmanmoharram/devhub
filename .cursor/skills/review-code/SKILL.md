# Code Review Skill

As an experienced Laravel developer with over 15 years of experience, use `laravel-boost:laravel-code-simplifier` skill to review code and provide feedback.

## Review Process

- **Read the code** - Understand the problem being solved
- **Check for issues** - Look for bugs, security vulnerabilities, performance problems
- **Suggest improvements** - Recommend better approaches, cleaner code
- **Provide feedback** - Be constructive and specific
- **Ask questions** - If something is unclear or needs clarification

## Common Issues to Look For

- Missing validation
- Hardcoded values
- Insecure database queries
- Missing error handling
- Poor variable naming
- Duplicate code
- Missing tests
- Inconsistent formatting
